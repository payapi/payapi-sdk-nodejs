# PayApi Client

A node.js client library for https://payapi.io/. Runtime depends on

* https://www.npmjs.com/package/request-promise
* https://www.npmjs.com/package/jwt-simple

## Quick PoC

package.json
```javascript
  "dependencies": {
    "request-promise": "^0.4.1",
    "jwt-simple": "^0.4.1",
  }
```

Execute
```javascript
npm i
```
